# Frontend Default Traps

Use this file when a first draft still feels generic after following `SKILL.md`.

## 1. Brandless SaaS Hero

**Signs**
- Centered headline
- Supporting paragraph
- Two CTA buttons
- Soft gradient blob
- Three feature cards underneath

**Why it fails**
- It can fit almost any product by swapping logo and copy.
- The composition carries no product truth.

**Better moves**
- Build the layout around how the product feels: editorial spread, control room, catalog wall, poster grid, notebook, instrument panel, etc.
- Make one structural decision memorable before polishing details.

## 2. Palette by Cliche

**Signs**
- Purple, pink, or blue gradients used because they look "modern"
- Clean white or dark navy background with no contextual reason

**Why it fails**
- It signals AI default taste instead of product-specific judgment.

**Better moves**
- Pull color logic from brand, industry, material, environment, or emotional tone.
- Pick a dominant color family and one or two accents with clear roles.

## 3. Typography by Habit

**Signs**
- Inter, Roboto, Arial, or Space Grotesk selected automatically
- No meaningful contrast between display and body styles

**Why it fails**
- The page sounds like the model, not the product.

**Better moves**
- Choose a type pairing with a reason.
- If the product already uses a standard font, create voice through scale, weight, tracking, line length, casing, and hierarchy instead of swapping fonts carelessly.

## 4. Safe Token Soup

**Signs**
- Same radius everywhere
- Same shadow everywhere
- Same padding rhythm everywhere
- Every container looks like the same card

**Why it fails**
- The interface becomes polished but anonymous.

**Better moves**
- Choose a surface language: razor-sharp, paper-like, soft and inflated, industrial framed, layered translucent, etc.
- Let borders, shadows, radii, and spacing reinforce that language consistently.

## 5. Decorative Motion

**Signs**
- Every element fades up
- Every button scales on hover
- Motion exists everywhere but says nothing

**Why it fails**
- Animation becomes garnish instead of direction.

**Better moves**
- Pick one motion idea that matches the concept: sweep, shutter, ticker, drift, reveal, mechanical snap, elastic bounce, etc.
- Use motion to establish rhythm and priority, not to decorate every element equally.

## 6. Section Cloning

**Signs**
- Every section is another grid of cards
- The whole page has one pacing speed

**Why it fails**
- Nothing stands out because everything is built from the same block.

**Better moves**
- Alternate dense and quiet sections.
- Mix oversized statements, narrow columns, image-led breaks, utility bands, or asymmetrical compositions.

## 7. Maximalism Without Hierarchy

**Signs**
- Many gradients, textures, glows, and animations
- No clear visual order

**Why it fails**
- Richness turns into noise.

**Better moves**
- Limit the number of loud ideas.
- Decide what leads, what supports, and what stays quiet.

## 8. Minimalism Without Craft

**Signs**
- Lots of whitespace, but weak proportions
- Neutral colors and thin lines doing all the work

**Why it fails**
- It reads as unfinished instead of refined.

**Better moves**
- Tighten spacing relationships.
- Improve type scale, alignment, and contrast.
- Make restraint feel deliberate, not empty.

## Questions to Unstick a Draft

- If the logo disappeared, would this still feel tied to the same product?
- What would the cheapest version of this design look like, and am I accidentally building that?
- Which single decision is doing the most work?
- What is memorable here besides polish?

## Last Pass Before Shipping

- Remove one cliche.
- Strengthen one contrast.
- Make one hero decision impossible to confuse with a template.
